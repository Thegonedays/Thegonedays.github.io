（github+hexo框架）免费博客的搭建

发表于 2018-03-20 | 分类于 搭建

一些技术性问题我这里就不啰嗦阐述了，具体细节参考diygree作者的简书教程这里我只是解决一下搭建过程中常见的一些问题。

失败了多次

  • hexo的安装：下载hexo时会遇到部分过程会被墙，因此将npm源换成淘宝镜像
  • 站点配置的问题：尤其注意deploy到github这一项，
  • type: git#使用git提交 respository: https://github.com/xxx/xxx.github.io.git#就是github的仓库地址这个尤其重要，否则hexo -d的提交会出现error not find git导致提交失败。
  • 注意站点文件配置中的格式问题：每一个对象的属性修改时前边需要加一个空格，否则会报错。
